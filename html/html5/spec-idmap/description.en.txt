idmap-r2255-r2256.txt
~~~~~~~~~~~~~~~~~~~~~

HTML5 specification id="" mapping table, between r2255 (CSS
postprocessor) and r2256 (anolis).

This file contains a tab (U+0009) separated list of r2255 and r2256
id="" values.

In r2256 some |dfn| element that had id="" specified in r2255 does not
have id="" specified.  In such cases the field for the r2256 id=""
value is left blank.
